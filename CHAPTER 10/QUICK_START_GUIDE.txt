PROBLEM: "object 'Hormone' not found" error
SOLUTION: This means the data loading has an issue. Try this:
          1. Run the test script: source("test_data_loading.R")
          2. This will diagnose the exact problem
          3. Check that column names are loaded correctly
          4. The updated Rmd file has better error handling

PROBLEM: "Data file not found"
SOLUTION: Check that HORMONE.DAT.txt exists at:
          Rosner_FundamentalsBiostatistics_8e_Data_Sets/ASCII-comma/HORMONE.DAT.txt

PROBLEM: "Package installation failed"
SOLUTION: 1. Check internet connection
          2. Run: install.packages("PACKAGE_NAME")
          3. Restart R and try again

PROBLEM: "Permission denied"
SOLUTION: 1. Run RStudio as Administrator
          2. Or save to a different directory

          - Multiple statistical tests

          Expect 2-5 minutes depending on your system.

================================================================================
STATISTICAL METHODS SUMMARY:
================================================================================

1. CHI-SQUARE TEST FOR INDEPENDENCE
   Purpose: Compare proportions across k groups
   H0: Proportions are equal across all groups
   H1: At least one proportion differs

2. FISHER'S EXACT TEST
   Purpose: Alternative when expected frequencies < 5
   Provides: Exact p-value

3. COCHRAN-ARMITAGE TREND TEST
   Purpose: Assess linear dose-response relationships
   H0: No linear trend across dose levels
   H1: Linear trend exists

4. TWO-PROPORTION Z-TEST (for sample size)
   Purpose: Calculate required sample size
   Inputs: Effect size, alpha, power
   Output: n per group
